# Улучшения алгоритма отбора быстрых серверов

## Что изменилось

Алгоритм отбора серверов теперь **приоритизирует стабильность** подключения, а не только скорость пинга. Это обеспечивает более надежное и предсказуемое соединение.

## Новые метрики

### 1. **Коэффициент вариации (CV%)**
- Формула: `(стандартное_отклонение / среднее) * 100`
- **Что это значит**: Процент вариативности пинга относительно среднего значения
- **Интерпретация**:
  - CV < 10% → Очень стабильный сервер ⭐⭐⭐
  - CV 10-20% → Стабильный сервер ⭐⭐
  - CV > 20% → Нестабильный сервер ⭐

### 2. **Spread (разброс)**
- Формула: `max_ping - min_ping`
- **Что это значит**: Максимальная разница между лучшим и худшим пингом
- **Единица**: миллисекунды (мс)

### 3. **Стандартное отклонение (stdev)**
- Формула: стандартная статистическая мера разброса
- **Что это значит**: Среднее отклонение значений пинга от среднего
- **Единица**: миллисекунды (мс)

### 4. **Процент успехов (success_rate)**
- **Что это значит**: Доля успешных попыток подключения из всех попыток
- **Значение**: 0.0 (0%) до 1.0 (100%)

## Новый алгоритм ранжирования

Сервера теперь сортируются по **взвешенному скору**:

```
score = probe_ok * 1000 
      + normalized_ping * 10 
      + normalized_cv * 5 
      + success_penalty
```

### Приоритеты:
1. **Прохождение TLS проверки** (вес: 1000) — самый важный фактор
2. **Низкий пинг** (вес: 10) — быстрое соединение
3. **Низкая вариативность** (вес: 5) — стабильность
4. **Высокий процент успехов** (штраф: 0-50 points) — надежность

## Новые флаги командной строки

### `--max-cv <процент>`
Фильтрует серверы с высокой вариативностью пинга.

**Примеры:**
```bash
# Использовать только очень стабильные серверы (cv < 10%)
python3 tools/vpn_picker.py --max-cv 10

# Использовать стабильные серверы (cv < 25%)
python3 tools/vpn_picker.py --max-cv 25
```

### `--max-ping-spread <мс>`
Фильтрует серверы, где разница между max и min пингом превышает указанное значение.

**Примеры:**
```bash
# Использовать только серверы с разбросом < 1мс
python3 tools/vpn_picker.py --max-ping-spread 1.0

# Использовать серверы с хорошей стабильностью (разброс < 2мс)
python3 tools/vpn_picker.py --max-ping-spread 2.0
```

## Рекомендации по использованию

### Для максимальной стабильности:
```bash
python3 tools/vpn_picker.py \
  --max-cv 15 \
  --max-ping-spread 1.5 \
  --tries 5 \
  --min-successes 4
```

### Для баланса скорости и стабильности:
```bash
python3 tools/vpn_picker.py \
  --max-cv 25 \
  --max-ping-spread 3.0 \
  --tries 3
```

### Для минимальной задержки (без строгой фильтрации):
```bash
python3 tools/vpn_picker.py \
  --tries 3
```

## Файлы отчетов

- **report.json** — полный отчет с метриками каждого сервера и информацией о стабильности
- **ping_top.tsv** — таблица лучших серверов с новыми колонками стабильности
- **best.txt** — лучший сервер с учетом новых критериев
- **top.txt** — топ-N серверов (по умолчанию 10)
