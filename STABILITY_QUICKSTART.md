# Quickstart: Улучшенный алгоритм выбора серверов

## 🚀 Быстрый старт

### Вариант 1: Самые надежные серверы
```bash
python3 tools/vpn_picker.py --max-cv 10
```
Выведет только серверы с минимальной вариативностью пинга.

### Вариант 2: Оптимальный баланс
```bash
python3 tools/vpn_picker.py --max-cv 25 --tries 5
```
Выведет быстрые И стабильные серверы на основе 5 пингов.

### Вариант 3: Максимальная скорость
```bash
python3 tools/vpn_picker.py
```
Выведет просто самые быстрые (без фильтра стабильности).

## 📊 Что означают метрики?

| Метрика | Что это | Пример |
|---------|--------|--------|
| **ping_ms** | Медиана пинга | 1.5 мс |
| **cv%** | Вариативность пинга в % | 5.3% = очень стабильный |
| **spread** | Разница max-min | 0.16 мс = стабильное |
| **stdev** | Стандартное отклонение | 0.08 мс = низкий разброс |

## ✅ Таблица интерпретации CV%

| CV% | Стабильность | Рекомендация |
|-----|-------------|--------------|
| < 10% | ⭐⭐⭐ Отлично | Идеально для всех случаев |
| 10-20% | ⭐⭐ Хорошо | Нормально для streaming |
| 20-30% | ⭐ Приемлемо | Может быть нестабильным |
| > 30% | ❌ Плохо | Избегать |

## 🎯 Примеры реальной жизни

### Нужна максимальная стабильность?
```bash
python3 tools/vpn_picker.py --max-cv 10 --max-ping-spread 1.0 --tries 10
```

### Нужна хорошая скорость и стабильность?
```bash
python3 tools/vpn_picker.py --max-cv 20 --tries 5
```

### Просто найти самый быстрый?
```bash
python3 tools/vpn_picker.py --tries 3
```

## 📁 Результаты

После запуска скрипта в папке `local-out/` появляются:

- **ping_top.tsv** — таблица с ВСЕМИ метриками (открывай в Excel!)
- **report.json** — полная информация в JSON формате
- **best.txt** — лучший найденный сервер
- **top.txt** — топ-10 серверов

## 🔧 Все доступные флаги

```bash
python3 tools/vpn_picker.py --help
```

Главные новые флаги:
- `--max-cv <число>` — максимальный коэффициент вариации (%)
- `--max-ping-spread <мс>` — максимальный разброс пингов (мс)
- `--tries <число>` — количество попыток пинга (по умолчанию 3)
- `--min-successes <число>` — минимум успешных попыток

## 💾 Использование в скриптах

```bash
# В update_and_pick.sh уже включены оптимальные настройки
bash tools/update_and_pick.sh

# Или с собственными параметрами
MAX_CV=15 python3 tools/vpn_picker.py --max-cv 15 --tries 5
```
