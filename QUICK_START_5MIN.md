# ⚡ Быстрый старт: 5 минут до VPN

## 🎯 За 5 минут вы будете в VPN

### Вариант 1️⃣: На Windows/Mac (самый легкий)

**Шаг 1: Скачать Clash** (3 минуты)
- Windows: Clash.exe
- Mac: ClashX

**Шаг 2: Активировать подписку** (1 минута)
- В Telegram боте: `/subscribe 7`
- Получаете ссылку: `http://localhost:8080/sub/123456/TOKEN.txt`

**Шаг 3: Добавить в Clash** (1 минута)
1. Откройте Clash
2. Profiles → **+**
3. Вставьте ссылку
4. Нажмите ✅

**Шаг 4: Подключиться** (30 секунд)
1. Выберите сервер из списка
2. Нажмите **Connect**
3. ✅ Готово!

## ❌ Частые ошибки

### "404 Not Found"
```
❌ Ссылка неправильно скопирована
→ Скопируйте снова из сообщения бота
→ Проверьте что нет пробелов в начале/конце
```

### "Connection refused"
```
❌ Сервер не работает
→ Попробуйте позже
→ Проверьте статус: /status
```

### "Invalid subscription"
```
❌ Ссылка не работает в клиенте
→ Попросите новую: /link
→ Установите более новую версию клиента
```

### "Subscription expired"
```
❌ Подписка истекла
→ Активируйте новую: /subscribe 30
```

## 🔒 Важные правила

1. **Ссылка = приватная**
   - Не передавайте никому
   - Это ваш персональный доступ

2. **Обновляется автоматически**
   - Не нужно переходить по ссылке
   - Клиент сам все обновляет

3. **Работает только с активной подписой**
   - После истечения = 403 Forbidden
   - Нужна новая подписка

4. **Выбирайте сервер в клиенте**
   - Каждый сервер = отдельное подключение
   - Разные серверы = разные страны/скорости

## 💡 Советы

✨ **Лучшие сервера обычно:**
- Первые в списке
- С полосой пропускания 100 Мб/с+
- С пингом <100ms

⚡ **Для скорости:**
- Выбирайте ближайший к вам регион
- На мобилках включайте режим полета и выключайте
- Перезагружайте клиент если медленно

🔧 **Если не работает:**
1. Проверьте `/status` в боте
2. Попросите новую ссылку: `/link`
3. Обновите клиент до последней версии
4. Попробуйте другой клиент
