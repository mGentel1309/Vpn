# 🔗 Как использовать VPN ссылку пользователю

## 📋 Что получит пользователь

Когда пользователь отправит боту `/link` и в ответ получит:

```
🔗 Ваша персональная VPN ссылка

http://127.0.0.1:8080/sub/123456/rA7kH2pQx8bZ9mN1Y3tF5w.txt

📱 Как использовать:
1. Скопируйте ссылку выше
2. Откройте ваш VPN клиент
3. Добавьте подписку (Add/Import)
4. Вставьте ссылку
5. Готово! 🎉
```

---

## 🖥️ КОМПЬЮТЕР - Windows/Mac/Linux

### Вариант 1: Clash (Рекомендуется)

**Скачать:**
- Windows: https://github.com/ClashApp/Clash.For.Windows
- Mac: https://github.com/yichengchen/clashX
- Linux: https://github.com/Dreamacro/clash

**Использование:**
1. Откройте **Clash**
2. Menu → **Profiles** → **+**
3. Вставьте ссылку VPN:
   ```
   http://127.0.0.1:8080/sub/123456/rA7kH2pQx8bZ9mN1Y3tF5w.txt
   ```
4. Нажмите **Download**/**OK**
5. Выберите конфиг и нажмите **Use** / **Connect**

---

### Вариант 2: V2Ray / Xray

**Mac/Linux:**
```bash
# Скачать V2Ray
wget https://github.com/v2fly/v2ray-core/releases/download/v5.0.0/v2ray-macos-64.zip
unzip v2ray-macos-64.zip
cd v2ray-core

# Запустить с конфигом
./v2ray -c config.json
```

**Windows:**
1. Скачать V2Ray с https://github.com/v2fly/v2ray-core
2. Распаковать
3. Отредактировать `config.json` с конфигами
4. Запустить `v2ray.exe`

---

### Вариант 3: sing-box

**Скачать:** https://github.com/SagerNet/sing-box

**Использование:**
1. Откройте sing-box GUI
2. Connections → **+**
3. Вставьте ссылку
4. Connect

После добавления ссылки в клиент:
- ✅ Конфиги **автоматически** загружаются из нашего сервера
- ✅ Обновляются **каждый час** автоматически
- ✅ Все **(N серверов)** будут доступны в одном профиле
- ✅ Нажимайте **Connect** для использования VPN

## 🛠️ Если ссылка не работает

### Проблема: "Connection refused"
- Убедитесь что бот/сервер работает
- Проверьте `http://127.0.0.1:8080/health`

### Проблема: "Invalid subscription"
- Проверьте что подписка активна: `/status`
- Токен может быть устаревшим, попросите новую ссылку: `/link`

### Проблема: "Network error"
- Попробуйте другой VPN клиент (Clash, V2Ray и т.д.)
- Убедитесь что ссылка скопирована **полностью** без пробелов
